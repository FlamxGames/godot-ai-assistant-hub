# Support this project
Ways to support this project:
### 1. Star it in GitHub & give it a thumbs up in Godot Asset Store
  Helps to give this tool visibility.
### 2. Vote for what you care
  Check the [Issues section](https://github.com/FlamxGames/godot-ai-assistant-hub/issues) and vote up or down to give your opinion on them.
### 3. Report bugs
  If you find bugs, report them by creating new GitHub issues, make sure to mention the plugin version you are using, LLM provider, and any other relevant details.
### 4. Enhance it
  Discuss some enhancement and make a pull request. Check the [Contributing Guidelines](https://github.com/FlamxGames/godot-ai-assistant-hub/blob/main/CONTRIBUTING.md).
